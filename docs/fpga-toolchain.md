# FPGA Toolchain

Install FPGA toolchain mostly via
[lushaylabs](https://learn.lushaylabs.com/os-toolchain-manual-installation/)

Except:

 * Use `pipx` to setup a venv
 * Install [nextpnr-himbaechel](https://github.com/YosysHQ/nextpnr#nextpnr-himbaechel)
 * ...

Run `\bin\attach-to-wsl.cmd` from the Windows side to give WSL access to the
Tang Nano 9k USB device.
